## What's covered under DevOps & CI/CD

The engineering side of getting code from a developer's laptop to production reliably.

### Core areas

- **Pipelines** — GitHub Actions, GitLab CI, AWS CodePipeline patterns
- **GitOps** — ArgoCD, Flux, drift detection
- **Observability** — Prometheus, Grafana, Datadog, distributed tracing
- **SRE** — error budgets, SLOs, incident response
- **Release management** — canary, blue/green, feature flags
- **Automation** — self-service platforms, internal developer portals

### Working assumption

Every deployment should be a non-event. If it isn't, the pipeline is wrong — not the humans in the loop.
