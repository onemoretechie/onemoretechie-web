## What's covered under AWS

Services, serverless and cost optimisation on Amazon Web Services — real configurations from real production environments.

### Core areas

- **Compute** — EC2, Fargate, Lambda, App Runner
- **Storage** — S3, EFS, EBS, Glacier
- **Networking** — VPC, PrivateLink, Transit Gateway, Route 53
- **Data** — RDS, Aurora, DynamoDB, ElastiCache
- **Identity** — IAM, SSO, permission boundaries
- **Serverless** — Lambda, Step Functions, API Gateway, EventBridge
- **Cost & FinOps** — right-sizing, RIs, Spot, tagging strategy

Marketing content, service intros, or anything you can read on the AWS docs page. This is the working practitioner's view.
